## 2.4 The `.Git` Folder

if you want to see your .Git folder you need to type

```bash
ls -a 

## which shows all folders even the hidden ones
```

<aside>
ðŸ’¡ If you delete the `.git` folder then this folder is no longer a repository and **ALL YOUR HISTORY WILL BE LOST**

</aside>
